Project: Hospital Appointment Booking Portal
Tech: ASP.NET Core Razor Pages + C#

Flow:
1. User ? /Appointment
   - Facility select karo (dropdown)
   - Speciality select karo (AJAX)
   - Doctor select karo (AJAX)
   - Slot select karo (Calendar)
   - Continue ?

2. User ? /PatientDetails
   - New Patient / Already Registered
   - Form fill karo
   - Mobile OTP verify karo
   - Review & Confirm ?

3. User ? /ReviewAndConfirmation
   - Appointment summary
   - Payment ?

Key Files:
- Services/     ? API calls handle karta hai
- Models/       ? API response structures
- ViewModule/   ? Page pe dikhne wala data
- Helpers/      ? Session management
- API/Clients/  ? HTTP client + URL builder
- Pages/        ? UI (Razor Pages)
